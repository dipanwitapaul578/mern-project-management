# MERN Project Management App

A Trello/ClickUp inspired app built with MERN stack (MongoDB, Express, React, Node.js).

## Current Features (Backend)
- User registration & login with JWT + httpOnly cookies
- Password hashing with bcrypt
- Protected routes (/me)
- Global error handling
- MongoDB Atlas integration

## Tech Stack (so far)
- Node.js + Express
- MongoDB + Mongoose
- JWT for auth
- bcrypt for passwords
- cors, cookie-parser

## Setup
1. Clone repo
2. `cd server && npm install`
3. Create `.env` with MONGO_URI and JWT_SECRET
4. `npm start`
